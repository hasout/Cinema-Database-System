# Cinema Database System

## Overview
A relational database system designed for managing cinema operations, including movies, customers, bookings, and screening schedules. The project focuses on applying database design principles and SQL implementation.

## Technologies
- SQL
- MySQL

## Features
- Relational database design
- Entity-Relationship Diagram (ERD)
- Database normalization
- SQL queries
- Data integrity

## Concepts Applied
- Primary Keys
- Foreign Keys
- Relationships
- Normalization
